# PlayerSync JAR Goes Here

Place the PlayerSync JAR file in this directory.

## Download

Get the latest PlayerSync from:
https://github.com/mlus-asuka/PlayerSync/releases

## Expected File

The build.gradle expects a file named:
- `playersync-1.20.1-2.1.4.jar`

Or you can update the version in `gradle.properties` and adjust the build.gradle dependency.

## Example

After downloading, your libs folder should look like:
```
libs/
└── playersync-1.20.1-2.1.4.jar
```
